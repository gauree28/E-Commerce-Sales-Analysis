## Project Overview
This project analyzes one year of e-commerce sales data to identify sales trends,
top-performing products, and profit insights.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Power BI
- DAX

## Key Features
- Data cleaning and preprocessing
- Revenue and profit analysis
- Monthly sales trend visualization
- Interactive Power BI dashboard

## Insights
- Electronics category generated the highest revenue
- Sales peaked during festive months
- Certain products contributed to maximum profit

## How to Run
1. Clone the repository
2. Run `sales_analysis.py`
3. Import CSV into Power BI for dashboard creation
